<h1 align="center">The Lupaxa Code Playground</h1>

**The Lupaxa Code Playground** contains experimental projects, prototypes and exploratory code from across **The Lupaxa Project**.

Unlike the other organisations within the ecosystem, the Code Playground is intentionally informal. It provides a place to experiment with new ideas, explore
unfamiliar technologies, build proof-of-concepts and work on projects that don't naturally belong anywhere else.

## Focus

**The Lupaxa Code Playground** may contain projects such as:

- Experimental tools and utilities
- Proofs of concept
- Technology evaluations
- Hackathon projects
- Friday afternoon coding experiments
- Learning and research projects
- Small utilities and one-off scripts
- Projects that don't yet have a permanent home

Some projects may eventually mature into standalone repositories or move into another organisation within **The Lupaxa Project** ecosystem, while many will
simply remain experiments that capture ideas, explore technologies or demonstrate new approaches.
